## 🌐 部署方式

### 方式1: Vercel (推荐，免费)

```bash

npm i -g vercel

# 2. 进入项目目录
cd ~/Desktop/AI神器工坊

# 3. 一键部署
vercel
```

### 方式2: Netlify (免费)

```bash
# 1. 安装 Netlify CLI
npm i -g netlify-cli

# 2. 进入项目目录
cd ~/Desktop/AI神器工坊

# 3. 一键部署
netlify deploy --prod
```

### 方式3: GitHub Pages (免费)

```bash
# 1. 创建 GitHub 仓库
# 2. 上传 index.html
# 3. 在 Settings → Pages 启用
```
